# Robotics Subsystem (Experimental)

**Status: Experimental** — See [STATUS.md](../../STATUS.md) for details.

This subsystem provides a ROS2 bridge for robotics integration (Unitree and other ROS2 robots).
It is wired into the main tRPC router but has not been validated against real hardware.

Do not rely on this subsystem for production workloads without hardware integration testing.
