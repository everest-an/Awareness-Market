# Awareness Network Community

Welcome to the Awareness Network community! This space exists to help contributors share latent‑vector capabilities, evaluation scripts, and ecosystem tooling.

## How to Participate

- **Share capabilities**: contribute capability metadata and examples via the capability manifest.
- **Improve evaluations**: add or refine evaluation scripts and metrics.
- **Help others**: answer questions and propose improvements in issues and discussions.

## Contribution Pathways

- Read the contribution guide: CONTRIBUTING.md
- Follow the code of conduct: CODE_OF_CONDUCT.md
- Submit improvements via pull requests.

## Community Goals

- High‑quality capability listings
- Transparent evaluation metrics
- Secure, reproducible tooling

Thank you for helping build an open AI capability ecosystem.
